# Projet Limousine

Site web pour une entreprise de location de limousines.

## Description

Ce projet contient un site web complet pour une entreprise de location de limousines, incluant :

- Page d'accueil avec présentation des services
- Galerie photos des véhicules
- Formulaire de réservation
- Informations de contact

## Technologies utilisées

- HTML5
- CSS3
- JavaScript
- PHP pour le backend

## Structure du projet

- `index.html` - Page d'accueil
- `galerie.html` - Galerie photos
- `assets/` - Ressources (CSS, JS, images)
- `images/` - Images des véhicules
- Divers scripts PHP pour la gestion

## Installation

1. Cloner le repository
2. Placer les fichiers sur un serveur web (Apache recommandé)
3. S'assurer que PHP est activé

## Contact

Pour plus d'informations, contacter l'entreprise via le formulaire de contact sur le site.
